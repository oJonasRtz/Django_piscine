# Resume in HTML/CSS

This project aims to practice `HTML` and `CSS` focussing on semantics and some commun tags

- Lists: `<ul>`, `<ol>`, `<li>`
- Tables: `<table>`, `<th>`, `<tr>`, `<td>`
- Styling: `<style>` and basic `CSS` properties
- Links: `<a>`

It includes sections like:

- Personal info (name, title, links)
- Description
- Skills (programming languages, frameworks, tools)
- Education
- Experience
- Footer
