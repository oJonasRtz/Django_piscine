# 🏊 Django Piscine – 42

This repository contains my work from the Django Piscine at 42.

The piscine is an intensive backend training program focused on building strong fundamentals in Django, web architecture, and database integration.  
Each module introduces new concepts and increases in complexity, following the peer-to-peer methodology of 42.
